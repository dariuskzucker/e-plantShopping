# Paradise Nursery Shopping Cart Application

Paradise Nursery is a React and Redux based e-commerce application for an online plant nursery.

## Features

* Landing page with company branding
* About Us section
* Product listing page
* Three plant categories
* Shopping cart functionality
* Add to Cart button disabling
* Dynamic cart item count
* Quantity increase/decrease controls
* Item removal
* Total cart calculations
* Checkout placeholder

## Technologies Used

* React
* Redux Toolkit
* React Router DOM
* CSS

## Project Structure

* App.jsx
* App.css
* AboutUs.jsx
* ProductList.jsx
* CartItem.jsx
* CartSlice.js
